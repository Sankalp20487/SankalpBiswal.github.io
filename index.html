Projects

Below are the my Data Analytics projects on SQL, Python, R, & ML.


Text-to-Sql LLM Chatbot | University Explorer Chatbot

Created a chatbot to help students find up to date information regarding colleges in the US. This is a RAG implemetation of chatbot which converts Natural language into complex Sql queries to retrieve data from government websites such as ipeds.


FIFA Analytics using Machine Learning

Predicted player performance ratings using Lasso Regression, Classified player position based on performace attributes using KNN with Bagging, Tested age vs pace using H-Testing, and Predicted market values using Stepwise Linear Regression


Chicago Traffic Crash Analysis Using Big Data

Leveraged Lambda Architecture to analyze 1.85 million traffic crash records from Chicago. Using Databricks, Apache Spark, and Delta Lake, a data pipeline was built for streaming and batch processing. The results are visualized in an interactive dashboard, offering insights on crash locations, weather, and injury severity to guide traffic safety improvements.


Food Delivery App EDA | Zomato EDA

Conducted EDA on a dataset of different restaurants across 13 Indian metros, focusing on ratings, prices, and cuisines. Cleaned data, imputed missing values, and created visualizations.

Hire Musician


Designed the database schema and backend for a platform to simplify hiring and booking musicians. This project lays the groundwork for a seamless, user-friendly experience in artist discovery and management.

Financial Time Series Forecasting


This project involved forecasting the stock prices of Apple Inc. (AAPL), Honeywell (HON), and dry wine prices using time series models like exponential smoothing and ARIMA. I conducted statistical tests to validate the models, providing accurate predictions and insights into future price trends.


Profit Maximization Using Linear Programming

This project uses linear programming in Excel to maximize the monthly net profit of a hardware distribution center. By optimizing inventory levels within budget and space constraints, the analysis identifies the most profitable product mix and offers insights for increasing profit through budget or space expansion.
